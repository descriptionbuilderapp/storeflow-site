STOREFLOW WEBSITE — SPRINT 1

1. Extract this ZIP.
2. Copy the following folders into the root of your existing storeflow-site project:
   - app
   - components
   - public

3. Allow Windows/VS Code to replace existing files when prompted.

4. In the project terminal, run:
   npm run dev

5. Open:
   http://localhost:3000

Included routes:
- /
- /features
- /docs
- /support
- /privacy
- /terms

The Privacy Policy and Terms pages are placeholders and must be replaced with final commercial legal text before public release.
