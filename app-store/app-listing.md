# Extended Description

## Simplify wholesale operations

StoreFlow helps Shopify wholesale merchants manage purchasing rules, customer access, and wholesale ordering workflows from one unified platform.

Instead of combining multiple single-purpose apps, StoreFlow provides a centralized solution designed specifically for wholesale businesses.

---

## Manage purchasing policies

Configure wholesale purchasing requirements including:

- Minimum Order
- Product MOQ
- Variant MOQ
- Collection MOQ
- Saved Rule Sets

---

## Control customer access

Choose who can:

- View prices
- Add products to cart
- Complete checkout

Support customer tag-based purchasing rules and optional checkout validation.

---

## Improve the ordering experience

Provide wholesale customers with:

- Quick Order
- Consistent purchasing validation
- Streamlined ordering workflows

---

## Built for Shopify

StoreFlow is built using modern Shopify technologies, including Theme App Extensions and Shopify Functions where appropriate.

Designed to integrate naturally with Shopify while remaining easy to configure and maintain.
