# StoreFlow Website Rules

## Project

StoreFlow Official Website

## Purpose

This repository contains the official marketing website and documentation for StoreFlow.

It is not the Shopify App source code.

## Technology

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Vercel

## Development Principles

- Keep the website simple and fast.
- Prefer reusable components.
- Avoid unnecessary dependencies.
- Follow accessibility best practices.
- Optimize for SEO.
- Mobile-first responsive design.

## Design Style

- Modern SaaS
- Clean
- Minimal
- Shopify-inspired
- Consistent with the StoreFlow brand

## Pages

- Home
- Features
- Documentation
- Support
- Privacy
- Terms
