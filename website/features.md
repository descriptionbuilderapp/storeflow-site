# Features

StoreFlow brings together the essential tools wholesale merchants need to manage purchasing policies, customer access, and ordering workflows from one unified platform.

---

# Order Policies

Create and enforce purchasing rules that match your wholesale business.

### Minimum Order

Require a minimum cart value before checkout.

### Product MOQ

Set minimum quantities for individual products.

### Variant MOQ

Apply quantity requirements to specific product variants.

### Collection MOQ

Create purchasing rules for entire collections.

### Saved Rule Sets

Group multiple rules into reusable configurations.

---

# Customer Access

Control who can view prices and purchase products.

### Customer Tag Rules

Apply different purchasing policies based on customer tags.

### Hide Prices

Hide pricing from unauthorized visitors.

### Hide Add to Cart

Prevent purchasing until customers are eligible.

### Hide Buy Now

Disable accelerated checkout when required.

### Optional Checkout Block

Prevent checkout until purchasing requirements are met.

---

# Purchasing Experience

Improve wholesale ordering efficiency.

### Quick Order

Allow customers to order products faster using a streamlined ordering experience.

### Checkout Validation

Validate purchasing requirements before checkout.

---

# Operations

Manage wholesale settings from a centralized dashboard.

### Dashboard

View and manage your StoreFlow configuration in one place.

---

# Built for Shopify

StoreFlow is designed to work with Shopify using modern platform capabilities, including Theme App Extensions and Shopify Functions where appropriate.

---

# Why StoreFlow

Instead of combining multiple single-purpose apps, StoreFlow provides one unified platform for managing wholesale operations.

This reduces configuration complexity, improves consistency, and makes wholesale management easier to maintain.
